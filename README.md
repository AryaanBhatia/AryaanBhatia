I graduated from **UNSW** with a degree in **Artificial Intelligence**, where I built a strong foundation in machine learning, probabilistic reasoning, and intelligent systems. Now, in my Masters, I'm going deeper into the mathematical machinery that underpins it all — **optimisation, numerical methods, dynamical systems, and statistical modelling**.

The more I learn, the more I see how much of the world is waiting to be modelled, understood, and improved. I want to be someone who helps build that future — one well-reasoned system at a time.
